# Provident India

This repository contains a static website that visually replicates the homepage and basic layout of [www.providentindia.com](https://www.providentindia.com/).

## Tech Stack

- HTML5
- CSS3
- Basic JavaScript (for menu toggling, etc.)

## Structure

```
Provident-India-Services/
│
├── index.html
├── style.css
├── script.js
├── assets/         # Images/icons (placeholders)
└── README.md
```

## Getting Started

1. Clone this repository.
2. Open `index.html` in your browser.

## Notes

- All images are placeholders. Replace them with your own assets as needed.
- This is a static site for educational and prototyping purposes.
